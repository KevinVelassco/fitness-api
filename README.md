# The Family Album API

## Installation

```bash
$ npm install
```
## Running the Database

```bash
$ docker-compose up -d
```

## Environment Variables

Clone the ```.env.template``` file and rename the copy to ```.env``` and add the appropriate values.

## Running the app

```bash
# development
$ npm run start:prod

# watch mode
$ npm run start:dev

# production mode
$ npm run start
```
